O que é o HTML?

HTML (Linguagem de Marcação de Hipertexto) é o código que, é usado na estrutura de uma página web e seu conteúdo. Sua estrutura pode ser constituida em paragráfo, em uma lista com marcadores ou em tabelas, imagens, áudio e vídeo

O HTML não é uma linguagem de programação e sim uma linguagem de marcação, usado como estrutura de seu conteúdo. Os elementos do HTML pode ser usado para estabelecer limites ou juntar-se a diferentes partes do seu conteúdo para que ele se mostre ou represente de determinada forma.



Estrutura básica

O documento HTML sempre inicia com o que chamamos de estrutura básica que são:

  • Definição do documento (DOCTYPE)

    • O doctype não é uma tag HTML, é uma instrução para o navegador e outros programas que podem ler seu site, o código encontrado é um código HTML. Assim os navegadores e outros programas sabem o que fazer para mostrar-se da melhor forma possível. É importante saber que o doctype é OBRIGATÓRIO e deve ser sempre a primeira linha do seu documento.

  • html

    • O elemento HTML (ou HTML root element) representa a raiz de um HTML ou XHTML documento. Todos os outros elementos devem ser descendentes desse elemento.

  • Cabeça (head)

    • Essa parte da estrutura não é apresentado no navegador web quando a página é mostrada. Ele abrange título, links para o CSS, links para favicon personalizado entre outros.

  • Corpo (body)

    • Refere ao documento em si, a informação com clareza para o usuário ou leitor do documento.Toda apresentação de entrada de dados também se aplica nesta seção do documento. Sua estrutura de página terá elementos semânticos, onde serão declarados e identificados cabeçalho, rodapé, conteúdo principal etc.



Atributos das tags

Um atributo envolve uma informação adicional que nós comunicamos para a nossa tag, para que ela consiga cumprir seu papel ou função de maneira eficaz.

São informações que passamos na Tag para que ela se comporte da maneira esperada. Existem atributos globais (que funcionam em todas as Tags) e específicos (que são direcionados para cada Tag, através de especificação).



Listas

As listas HTML permitem que os desenvolvedores da web agrupem um conjunto de itens relacionados em listas.

  • Lista não ordenada: Uma lista não ordenada começa com a tag <ul>. Cada item da lista começa com a tag <li> . Os itens da lista serão marcados com marcadores (pequenos círculos pretos) por padrão.
  1. Uma lista ordenada é também uma lista de itens. As listas de itens são marcadas com números. Uma lista ordenada começa com a tag <ol>. Cada item da lista começa com a tag <li>.


Links

Links são encontrados em quase todas as páginas da web. Os links permitem que os usuários cliquem de uma página para outra.Um link não precisa ser texto. Um link pode ser uma imagem ou qualquer outro elemento HTML.

No HTML, os links são definidos pela tag <a>. Dentro dessa tag incluímos o atributo href (Hypertext Reference), que é o endereço de destino do link. Dentro do conteúdo da tag <a>, incluímos então o texto ou elemento que servirá como redirecionador, ou seja, que ao ser clicado, executará a função de redirecionar para o endereço dentro do atributo href.
